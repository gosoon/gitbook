kubernetes api 总览

kubectl 的所有操作都是调用 kube-apisever 的 API 实现的，所以其子命令都有相应的 API，每次在调用 kubectl 时使用参数 -v=9 可以看调用的相关 API，例：

$ kubectl get node -v=9

以下为 kubernetes 开发中常用的 API：
